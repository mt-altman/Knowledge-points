查询和聚合的基础使用

初级

术语定义

查询 (Query)：在数据库中检索和筛选数据的操作，用于从数据集中获取满足特定条件的记录。

聚合 (Aggregation)：对数据集进行统计分析的过程，通过计算汇总值（如总和、平均值、最大值等）来提取数据洞察。

查询和聚合是数据处理的基础操作，几乎所有数据库系统都支持这些功能。

实现细节与优化

基本查询操作

-- SQL基本查询示例
SELECT 列名1, 列名2 
FROM 表名
WHERE 条件;

-- 示例：查询所有年龄大于18的用户
SELECT user_id, username, age 
FROM users 
WHERE age > 18;

基本聚合操作

-- SQL基本聚合示例
SELECT 聚合函数(列名)
FROM 表名
[GROUP BY 分组列];

-- 示例：计算每个部门的平均工资
SELECT department_id, AVG(salary) as avg_salary
FROM employees
GROUP BY department_id;

应用场景

电子商务

查询：获取特定价格范围内的产品

聚合：计算每个类别的销售总额

社交媒体

查询：检索特定用户的所有帖子

聚合：统计每日活跃用户数量

实际示例：销售数据分析

-- 查询：获取最近30天的高价值订单
SELECT order_id, customer_name, total_amount
FROM orders
WHERE order_date >= DATEADD(day, -30, GETDATE())
  AND total_amount > 1000
ORDER BY total_amount DESC;

-- 聚合：按地区统计销售额
SELECT region, COUNT(*) as order_count, SUM(total_amount) as total_sales
FROM orders
GROUP BY region
ORDER BY total_sales DESC;

常见问题与解决方案

问题：查询速度慢

解决方案：创建适当的索引，优化查询语句，避免使用SELECT *

问题：聚合操作内存消耗大

解决方案：限制结果集大小，使用分批处理，考虑使用物化视图
